## About the Project

This project was developed as part of a ReactJS task/assignment.
It focuses on building a clean multi-page interface using React components and provides Back and Forward arrow navigation to move between pages smoothly.

The project demonstrates:

* Component-based design in React
* State management using useState
* Simple page navigation without using React Router
* Clean UI structure and styling

## Navigation Logic

The application uses React state to track the current page.
Arrow buttons allow users to move forward and backward between pages.
The arrows are disabled when the user reaches the first or last page to avoid invalid navigation.
